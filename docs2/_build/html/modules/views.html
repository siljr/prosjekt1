Views¶

band_booking.views.index(request)¶

Creates a role specific user page

band_booking.views.login_authenticate(request)¶

Authenticates the user and login if the username and password are correct, else the user is redirected to the login page with an error

band_booking.views.login_page(request, error=None)¶

Displays the login page if the user is not logged in, else it redirects to the index page. Displays an informative error message if the error argument is set.

band_booking.views.logout(request)¶

Logs the user out, redirects to the login page
